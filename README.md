# yadmm - Yet Another Diva Mod Manager
A (currently barebones) mod manager for Project Diva MegaMix+ written in Electron and React. It's pronounced "yad-em-em".
> [!IMPORTANT]
> yadmm is developed primarily with Linux in mind. 
> However, Windows support is also a primary goal.
> If you encounter bugs on Windows, **please report them**!
---
## Currently Supported Features:
- Ability to enable/disable mods
- Ability to uninstall mods
- Ability to reorder mod priority
## Planned Additions:
- Ability to install from various services
- Ability to check for mod updates
- Ability to update/install DivaModManager
- Probably more that I'm forgetting
